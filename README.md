# Chronicles of Aryavarta

A browser-based RPG set in ancient India.

## Quick Start

Open `index.html` in a browser. No build step needed.

## Controls

- Arrow keys or WASD to move
- Spacebar to interact

## Concept

Explore a minimalist world inspired by ancient Indian aesthetics. Collect rudraksha malas, maintain dharma, manage wealth, and encounter random events.

## Tech

- Vanilla JS + Canvas
- No deps
- GitHub Pages ready

## Development

Commit and push; GitHub Pages will serve from `main` branch root.
